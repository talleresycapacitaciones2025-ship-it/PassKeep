# 🔐 PassKeep - Gestor de Contraseñas Seguro

**PassKeep** es una aplicación moderna y segura diseñada para gestionar todas tus contraseñas en un solo lugar. Protege tu información digital con una bóveda cifrada y genera contraseñas robustas al instante.

## 🚀 Características Principes

- **Bóveda Segura:** Almacenamiento cifrado de todas tus credenciales.
- **Generador de Contraseñas:** Crea contraseñas fuertes y personalizadas.
- **Interfaz Intuitiva:** Diseño limpio y fácil de usar basado en Material Design 3.
- **Acceso Rápido:** Desbloqueo seguro con contraseña maestra.
- **Código Abierto:** Transparente y auditables.

### 📱 Instrucciones de Instalación

1.  Haz clic en el botón de **Descargar** de arriba.
2.  Una vez finalizada la descarga, abre el archivo `app-debug.apk`.
3.  Si tu dispositivo muestra una advertencia de seguridad, ve a **Ajustes > Seguridad** y permite la **"Instalación de aplicaciones de fuentes desconocidas"** para tu navegador o gestor de archivos.
4.  Confirma la instalación y ¡listo! Ya puedes usar PassKeep.

## 🛠️ Tecnologías Utilizadas

- **Lenguaje:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Arquitectura:** MVVM (Model-View-ViewModel)
- **Base de Datos:** Room / SQLCipher (Cifrado)
- **Construcción:** Gradle
